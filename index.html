Welcome Errebody!

Have you ever wanted to learn Japanese? Have you ever want to visit Japan? Are you generally interested in Japan and Japanese culture in general? Well then this app is for you. Otaku is an open-source app that is built for anyone keen on learning Japanese.

iPhone, iPod touch, and iPad ready.

This app is built for iOS devices, with Android support coming soon. This app was made from the ground up, using web pages for most of the lessons. By using web pages, we can include many more things other than just the features of iOS devices.

Extra features!

By implementing HTML webpage instead of native objects we can include more style and design.
